# React UpTasks (Front)

## App de administracion de proyectos

### 🚀 Quick start

#### Instalar dependencias

```bash
npm install
```

#### Variables de entorno

```bash
Copiar las de .env.template y crear el archivo.env
```

#### Correr aplicación localmente

```bash
npm run dev
```
